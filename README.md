Libstack: Frontend
==================

Setup
-----

### Install Node Environment

```bash
$ curl -o- https://raw.githubusercontent.com/creationix/nvm/[LATEST_VERSION]/install.sh | bash
$ nvm install iojs
$ nvm alias default iojs
```

### Install/Update System Packages

```bash
$ brew install git git-flow-avh
```

### Init the Project

```bash
$ npm run init
```
